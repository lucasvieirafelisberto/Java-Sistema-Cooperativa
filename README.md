# 🤝 Sistema de Cooperativa em Java

Aplicação desenvolvida em **Java** para simular o funcionamento de uma cooperativa, permitindo o gerenciamento de cooperados e operações básicas via terminal.

---

## 🚀 Funcionalidades

- Cadastro de cooperados  
- Gerenciamento de dados dos membros  
- Registro de operações básicas  
- Listagem de cooperados cadastrados  
- Menu interativo via terminal  

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)  
- Uso de classes e objetos  
- Estruturas condicionais (`if/else`)  
- Laços de repetição (`for`, `while`)  
- Entrada e saída de dados  
- Organização de dados em memória  

---

## 🛠️ Tecnologias

- Java
